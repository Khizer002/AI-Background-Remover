<h1>AI Background Remover</h1>

<p>
This is a simple AI-based background remover built using Streamlit and the rembg library. 
The app allows you to upload an image and automatically removes its background using 
a pre-trained deep learning model.
</p>

<p>
I created this project to experiment with image processing and to understand how 
AI models can be integrated into a basic Python web app. It’s a small project, 
but it helped me explore how real-world AI tools actually work behind the scenes.
</p>

<h3>Features</h3>
<ol>
  <li>Upload image directly from your system</li>
  <li>Automatic background removal using AI</li>
  <li>Instant preview of original and processed image</li>
  <li>Simple and clean user interface</li>
</ol>

<h3>Tech Stack</h3>
<ul>
  <li>Python</li>
  <li>Streamlit</li>
  <li>rembg</li>
  <li>Pillow (PIL)</li>
</ul>

<h3>How to Run</h3>
<ol>
  <li>Clone this repository</li>
  <li>Install the required packages:</li>
</ol>

<pre><code>pip install streamlit rembg pillow</code></pre>

<h3>Run the app:</h3>

<pre><code>streamlit run main.py</code></pre>

<h2>Why This Project?</h2>

<p>
The goal of this project was to get some hands-on experience with AI-based image 
processing and understand how background removal tools work. It also helped me 
practice building simple interactive apps using Streamlit and managing external 
libraries inside a Python project.
</p>
